cseslib4py 文档
===================

 | 适用于 Python 的 CSES API

简介
----

欢迎来到 ``cseslib4py`` 的文档页面。

**cseslib4py** 是一个 Python 库，提供对 `CSES <https://github.com/SmartTeachCN/CSES/>`_ 格式的访问，同时具有简单易用的设计。

此外，它由 `MacrosMeng <https://github.com/MacroMeng>`_ 重构，以提供比已经停更的 `pycses <https://github.com/SmartTeachCN/pycses>`_ **更好** 和 **具有更高易读性** 的访问。

许可证
------
**cseslib4py** 采用 **MIT** 许可证。您可以在遵守许可证条款的前提下自由使用、修改和分发该库。
